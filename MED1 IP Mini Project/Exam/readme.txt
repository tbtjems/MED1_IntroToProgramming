# Drumming Pharaohs

Drumming Pharaohs is a visual and audiable programming made with Processing 3.5.3

## Usage

Choose the amount of drumming pharaohs in the naming of the Pharaoh instances:
Pharaoh [] myPharaohs = new Pharaoh[9];

Move mouse coursor around to move the sun, and enjoy a randomly created drumming performance by the pharaohs.
